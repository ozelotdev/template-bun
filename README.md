## Development

### Prerequisites

- Container Runtime (Docker, Podman, etc.)
- Development Container
- Visual Studio Code

### Getting Started

1. Clone the repository
1. Open the repository in Visual Studio Code
1. Reopen in Container
1. Run `bun install`
